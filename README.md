# andrealeopardi.com

This is my personal website hosted at <https://andrealeopardi.com>.

It's built with [Astro] and MDX. Deployed to GitHub Pages via Actions.

```bash
npm install
npm run dev        # http://localhost:4321
npm run build      # emits dist/
```

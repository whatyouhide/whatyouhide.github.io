## Use Disambiguation Stylistic Set for UI

Enable ss02 (or your font's disambiguation set) in code-facing UIs to distinguish I, l, 1 and 0, O.

**Correct:**

```css
.code-ui { font-feature-settings: "ss02"; }
```

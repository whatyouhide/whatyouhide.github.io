## Use text-wrap pretty for Body Text

Use text-wrap: pretty for body text to reduce orphans. Use text-wrap: balance for headings.

**Correct:**

```css
p { text-wrap: pretty; }
h1, h2, h3 { text-wrap: balance; }
```
